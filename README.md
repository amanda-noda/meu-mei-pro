# MEU MEI PRO - Controle Financeiro

Aplicativo de controle financeiro para Microempreendedores Individuais (MEI): lançamentos, painel de finanças e cadastro de receitas e despesas.

## Como rodar

```bash
npm install
npm run dev
```

Acesse no navegador o endereço indicado (ex.: `http://localhost:5173`).

## Build

```bash
npm run build
```

Arquivos de produção na pasta `dist/`.

## Tecnologias

- React 19 + TypeScript
- Vite 7
- Dados no `localStorage`
